# Project Title

Project with returns 5 images(It returns only images name, they're stored in laravel storage)

## How to install
1. Clone the project to your directory
2. Run composer install
3. Run cp .env.example .env
4. Run php artisan key:generate

### How to use
The application has only one endpoint : /api/doggos which returns 5 imagies

To test run php artisan serve to run local server and make HTTP GET request to http://127.0.0.1:8000/api/doggos

Or run DoggoTest.php
